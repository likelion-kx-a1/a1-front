## 화면 흐름도

주요 사용자 흐름을 순서대로 기술합니다.

### 1. 전체 화면 흐름 (Main Flow)

```
랜딩(작업 페이지)
      │  로그인
      ▼
   로그인
      │
      ▼
랜딩(로그인 상태)
      │
      ├──────────────┬──────────────┬──────────────┐
      ▼              ▼              ▼              ▼
 이미지 생성      영상 생성       라이브러리       마이페이지
```

### 2. Key Image 제작 흐름 (핵심 동선)

```
이미지 생성 화면
      │  텍스트 입력 + 상세 세팅
      ▼
 Key Image 생성
      │
      ▼
   결과 확인
      │
      ├─▶ 다운로드
      └─▶ 라이브러리 저장 (생성 자산 등록)
```

### 3. 영상 제작 흐름

```
라이브러리
      │  Key Image 검색 · 선택
      ▼
  영상 생성 화면
      │  Image + Prompt (또는 Text)
      ▼
 영상 생성 (비동기 · 진행 상태)
      │
      ▼
   완료 · 결과 확인
      │
      ├─▶ 다운로드
      └─▶ 라이브러리 저장
```

### 4. 역프롬프트 재생성

```
라이브러리 / 생성 화면
      │  이미지 선택
      ▼
  역프롬프트 (프롬프트 추출)
      │
      ▼
  프롬프트 수정
      │
      ▼
    재생성
```
